# alfred-scn-jira
> Alfred Jira Module


## Install

```
$ npm install -g alfred-scn-jira
```

*Requires [Node.js](https://nodejs.org) 4+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*


## Usage

In Alfred, type `jira`, <kbd>Enter</kbd>, and your query.
